Tech Talent & Market Intelligence Dashboard
An interactive, data-driven web application built with Python and Streamlit designed to analyze tech industry workforce trends, benchmark salary distributions, and perform multi-currency financial analysis for global talent market intelligence.

Live Demo: Tech Talent Market Intelligence App
Key Features
* Interactive Executive Dashboard: Clean user interface providing top-level KPI metrics, salary benchmarking, and market distribution summaries.

* Dynamic CSV Upload: Drag-and-drop custom datasets to auto-generate real-time analytics.

* Salary & Compensation Analysis: In-depth visual breakdowns across job roles, experience levels, and geographical regions.

* Multi-Currency Conversion: Interactive currency scaling to benchmark global compensation packages accurately.

* Filterable Data Views: Slice and dice datasets dynamically by role, tech stack, location, and seniority.
  
Tech Stack & Tools

Framework: Streamlit

Data Processing & Analytics: Pandas, NumPy

Deployment: Streamlit Community Cloud

Version Control: Git & GitHub

├── requirements.txt    # Python dependencies and libraries required

Local Installation & Setup
If you want to run this project locally on your machine:

1.Clone the repository:
git clone https://github.com/MehaMariamSony/Tech-Talent-Market-Intelligence.git
cd Tech-Talent-Market-Intelligence

2.Install dependencies:
pip install -r requirements.txt

3.Run the Streamlit application:
streamlit run app.py
